## ğŸ¯ Key Features Summary

### 1. **Dual Mode Operation**
- **Global Mode**: Open file sharing, no encryption, faster transfers
- **Private Room Mode**: Password-protected, encrypted, isolated

### 2. **Security Layers**
- Zero-Knowledge Proof authentication (no password transmission)
- AES-256-GCM encryption in rooms
- WebRTC DTLS encryption (always active)
- RSA identity verification

### 3. **OS Concepts**
- Download queue with multiple scheduling algorithms
- Semaphore for resource management (max 3 concurrent)
- Performance metrics (wait time, turnaround, throughput)

### 4. **P2P Architecture**
- Direct peer-to-peer file transfer
- No server relay (server only for signaling)
- NAT traversal with STUN servers
- Adaptive chunk sizing for stability

### 5. **User Experience**
- Drag-and-drop file sharing
- Real-time progress tracking
- QR code for mobile sharing
- Incognito mode for private rooms
- Network performance graphs
